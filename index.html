PsuedoEntity

A class for generating realistic looking data for people, companies, etc.

Installation

Add this line to your application's Gemfile:

gem 'psuedo_entity'

And then execute:

$ bundle

Or install it yourself as:

$ gem install psuedo_entity

Usage

entity = PseudoEntity.new
puts entity
